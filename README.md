## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/joacoderdev/tiendajoacoder.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the APP
   ```js
   npm start;
   ```
